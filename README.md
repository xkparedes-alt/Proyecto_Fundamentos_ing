Escuadrón Hero es un proyecto colaborativo desarrollado en Python cuyo objetivo es practicar la programación modular y el trabajo en equipo con Git y GitHub. Cada integrante del equipo crea un héroe diferente utilizando una clase en un archivo independiente (.py), junto con una instancia del objeto ya creada. El proyecto simula un entorno real de desarrollo, donde el código se construye de forma distribuida y luego se integra en un archivo principal.
El líder del equipo se encarga de integrar todos los módulos en main.py, importar los héroes y crear un menú interactivo en consola que permite seleccionar un personaje, mostrar sus estadísticas y ejecutar su acción especial. El flujo de trabajo incluye la creación de ramas, commits individuales y Pull Requests, asegurando una correcta integración y funcionamiento del programa al ejecutar python main.py.
